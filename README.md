# NetVision AI — Intelligent Network Monitoring & Threat Detection Platform

NetVision AI is an AI-powered network observability platform that captures real-time network traffic, analyzes protocols, visualizes network health, detects anomalies using machine learning, and generates intelligent security alerts. 

Built with industry-grade technologies, this tool serves as a lightweight alternative to enterprise solutions like Cisco Secure Network Analytics.

## Features
- **Monitoring**: Live packet capture, TCP/UDP/ICMP/DNS/HTTP analysis.
- **Analytics**: Bandwidth consumption, top talkers, traffic trends.
- **Security**: Port scan detection, DDoS detection, suspicious IP alerts.
- **AI Integration**: Traffic anomaly classification and smart alert prioritization.
- **Dashboard**: Real-time charts, live packet table, network health score.

## Architecture & Tech Stack
- **Backend**: Python, FastAPI, Scapy, Socket.IO
- **Frontend**: React, Tailwind CSS, Recharts, Framer Motion
- **Machine Learning**: Scikit-Learn, XGBoost, Pandas, NumPy
- **Database**: MongoDB, Redis
- **Deployment**: Docker, GitHub Actions
